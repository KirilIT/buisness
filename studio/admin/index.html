Як захостити на Netlify

Крок 1. Зареєструйтесь на netlify.com
Крок 2. «Add new site» → підключіть GitHub KirilIT/buisness
Крок 3. У налаштуваннях сайту вкажіть Base directory: studio (тільки FORM MEDIA, без списку клієнтів)
· Publish directory: .
· Functions: netlify/functions
· Build command: npm install (для Netlify Blobs)
Крок 4. Environment variables → додайте ADMIN_PIN = ваш секретний PIN
Крок 5. Deploy. Сайт: https://ваш-сайт.netlify.app, адмінка: /admin/
Локально з Netlify Functions:
npm install
npx netlify dev
Сайт і API працюють як на продакшені.

Без Netlify (швидкий тест):
node admin/server.mjs → сайт і адмінка на localhost:3947
